# @dagu-org/boltbase-linux-armv7

Platform-specific binary package for Boltbase.

This package contains the pre-compiled Boltbase binary for Linux ARMv7 (32-bit ARM).

## Installation

**Do not install this package directly!**

This package is automatically installed as an optional dependency when you install the main package:

```bash
npm install @dagu-org/boltbase
```

## About

This is a platform-specific distribution of [Boltbase](https://github.com/dagu-org/dagu), a powerful workflow orchestration engine.

For documentation and usage, please see the main package: [@dagu-org/boltbase](https://www.npmjs.com/package/@dagu-org/boltbase)
