# @dagucloud/dagu-linux-x64

Platform-specific binary package for Dagu.

This package contains the pre-compiled Dagu binary for Linux x64.

## Installation

**Do not install this package directly!**

This package is automatically installed as an optional dependency when you install the main package:

```bash
npm install @dagucloud/dagu
```

## About

This is a platform-specific distribution of [Dagu](https://github.com/dagucloud/dagu), a powerful workflow orchestration engine.

For documentation and usage, please see the main package: [@dagucloud/dagu](https://www.npmjs.com/package/@dagucloud/dagu)
